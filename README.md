# My Version of Yelp Camp
 My Version of Yelp Camp. This is based on the Web Development Bootcamp 2021 by Colt Steele from Udemy
 Tech stack used:
 1. Express JS
 2. EJS & EJS Mate
 3. Bootstrap 5
 4. MongoDB and Mongoose
 5. JOI Validator
 6. Express Passport
 7. Node.js
 8. Cloudinary
 9. JsDelivr
 10. MapboxGL JS
 11. Heroku
